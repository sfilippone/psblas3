free -- Free a preconditioner 

call prec%free(info)

Type:
Asynchronous.
On Entry
prec
the preconditioner.
Scope: local.
Type: required
Intent: inout.
Specified as: a preconditioner data structure precdatapsb_prec_type.
On Exit

prec
Scope: local
Type: required
Intent: inout.
Specified as: a preconditioner data structure precdatapsb_prec_type.
info
Scope: global
Type: required
Intent: out.
Error code: if no error, 0 is returned.
Notes Releases all internal storage.
