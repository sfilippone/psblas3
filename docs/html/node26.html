get_size -- Get maximum number of nonzero elements in a sparse matrix 

maxnz = a%get_size()

Type:
Asynchronous.
On Entry
a
the sparse matrix
Scope: local

On Return
Function value
The maximum number of nonzero elements that can be stored in sparse matrix a using its current memory allocation.
