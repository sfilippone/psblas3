psb_is_local -- 

call psb_is_local(x, desc_a)

Type:
Asynchronous.
On Entry
x
Integer index.
Scope: local
Type: required
Intent: in.
Specified as: a scalar integer.
desc_a
the communication descriptor.
Scope:local.
Type:required.
Intent: in.
Specified as: a structured data of type descdatapsb_desc_type.

On Return
Function value
A logical mask which is true if is local to the current process Scope: local
Type: required
Intent: out.

Notes

  1. This routine returns a .true. value for an index that is local to the current process, including the halo indices
