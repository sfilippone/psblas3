get_diag -- Get main diagonal 

    call a%get_diag(d,info)

Returns a copy of the main diagonal.

Type:
Asynchronous.
On Entry
a
the sparse matrix.
A variable of type psb_Tspmat_type.
Scope: local.
On Return
d
A copy of the main diagonal.
A one-dimensional array of the appropriate type.
info
Return code.
